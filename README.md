
# Python Projects

A collection of small Python projects built while learning the language - CLI tools focused on core concepts like loops, data structures, and error handling.

## Projects
### 1. To-Do List (`to-do.py`)
A command-line task manager. Add, view, and delete tasks from a running list.
### 2. Expense Tracker (`expense-tracker.py`)
Logs multiple expense entries and calculates the running total. 
### 3. Random Password Generator (`random-password-generator.py`)
Generates a randomized password based on user-defined criteria.
### 4. Text Analyzer (`text-analysis.py`)
Analyzes a block of text — counts total words, finds the most frequently used word, and calculates average word length.
